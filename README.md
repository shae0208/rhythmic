# Rhythmic Music Bot
A comprehensive music bot with youtube query integration

<img src="https://github.com/shae0208/rhythmic/blob/main/icon/icon.png" alt="Rhythmic icon" width="500" height="500">

## function
Includes functionality for:
* /play
* /pause
* /resume
* /skip
* /stop

## dependencies
To install dependencies in python3:

    pip install -r requirements.txt
